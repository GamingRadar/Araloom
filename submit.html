Share your template

Built a useful template in Araloom? Share it and let others download it in one click.

How submission works

1

Build your template in Araloom

Create a template inside the extension with the fields you need, text, tags, select, dates, etc.

2

Export it as a .json file

In Araloom, go to your template → Export. You'll get a JSON file matching the schema below.

3

Submit via the form or GitHub

Use the form on the right to email it to us, or open a GitHub Issue if you prefer. We review and publish it, usually within 48 hours.

4

It appears in the gallery

Once live, anyone can download your template directly from the gallery. Your name appears as author.

Expected JSON format

// Your exported template will look like this:
{
  "id": "tab_xxxxxxxx_xxxxxxx",
  "name": "Your Template Name",
  "description": "What it captures and who it's for",
  "icon": "📚",
  "color": "#6366f1",
  "fieldSchema": [
    {
      "id": "tab_xxxxxxxx_xxxxxxx",
      "label": "Title",
      "type": "auto_title",
      "required": true
    },
    {
      "id": "tab_xxxxxxxx_xxxxxxx",
      "label": "Notes",
      "type": "textarea",
      "placeholder": "Your notes here...",
      "required": false
    }
  ],
  "createdAt": "2026-01-01T00:00:00.000Z",
  "updatedAt": "2026-01-01T00:00:00.000Z"
}

Submission guidelines

✅

Template should have a clear, specific purpose, not just "general notes"

✅

Include at least one auto_url field so source is always saved

✅

Write a description that explains what it captures and who it's for

❌

No duplicates, check the gallery before submitting a similar template

❌

No personal or sensitive data embedded in the JSON
