How Araloom works

AraLoom turns anything you read into structured, connected data without leaving the page.

01

Highlight to Capture

Select any text on a website and the extension activates instantly. It grabs what you need, along with the source details, automatically.

Smart Capture:
  • Selected text or images
  • Website URL & Page Title
  • Automatic timestamp

02

Organize with Fields

Your clips flow into smart forms. Instead of a messy pile of notes, you get structured entries that are easy to search and sort.

Common Fields:

  • Select for status or priority

Reference

Template field types

Every field in a template JSON maps to one of these types.

auto_title

Automatically captures the page's <title> tag. No typing required.

auto_url

Automatically saves the full URL of the page being captured.

text

Single-line text input. Good for names, companies, short values.

textarea

Multi-line text. Use for notes, summaries, takeaways. Supports @mentions.

tags

Comma-separated tag chips. Great for categorizing and filtering entries.

select

Dropdown with predefined options. Useful for status fields like Applied/Rejected.

bibtex

Structured citation block with sub-fields: author, year, journal, doi, title.

date

Date picker. Great for deadlines, publication dates, follow-up reminders.

number

Numeric input. Use for salaries, prices, scores, quantities.
